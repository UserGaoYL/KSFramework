工程打包方法

  1. 打包前，请先 Build Assetbundle

  2. 点击菜单栏的KEngine - Autobuilder - Build Android或(其它平台)

不建议使用File - Build Settings - Build，因为Autobuilder 内部会处理一些操作，比如

  1. 自动生成xlua代码
  2. 自动拷贝Product目录下的资源到StreamingAssets目录下
  3. 并且生成的安装包会自动放在Product/Apps/Android或Product/Apps/Windows下，
  4. 整个过程是自动化的，可和Jenkins等自动化打包平台集成。

Link 资源

打包一次后，在StreamAsset目录会Link product的资源，Link之后两边的资源是同步的

可以通过KEngine - Symbol Link Resources 来进行重新Link或取消
